# 🛒 E-Commerce & Salesforce Customer Sales Analysis

## 📌 Business Overview
This project analyzes E-commerce transaction data and Salesforce CRM metrics to extract actionable insights regarding customer buying behavior, sales funnels, and revenue trends.

## 🛠️ Tools Used
- **SQL (MySQL/PostgreSQL):** Data extraction, aggregations, and RFM Segmentation.
- **Python (Pandas, Seaborn):** Exploratory Data Analysis (EDA) & Data Cleaning.
- **Power BI / Tableau:** KPI Dashboards & Visualizations.

## 📊 Key Insights Derived
1. **Customer Segmentation (RFM Analysis):** Categorized customers into VIP, Loyal, and At-Risk segments.
2. **Sales Trends:** Identified peak purchasing periods and high-converting sales pipeline channels.
3. **Revenue Impact:** Top 20% of loyal customers contribute to 60%+ of overall sales revenue.

## 📁 Project Structure
- `sql_queries.sql` -> SQL Data Extraction & RFM Logic
- `sales_analysis.py` -> Python Data Analysis Script
